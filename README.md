# 运行

## shell法

- 在根目录运行shell

  ~~~shell
  npx http-server
  ~~~

  

- 浏览器输入默认地址（比如：http://127.0.0.1:8080）

## vscode法

- 安装对应插件live server
- 打开index.html右击运行open with live server
